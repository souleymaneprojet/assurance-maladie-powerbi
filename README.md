# 📊 Dashboard Power BI - Analyse de la performance (Assurance Maladie)

## 🎯 Objectif

Ce projet vise à analyser la performance opérationnelle et la qualité de service à partir de données de santé simulées.

## 📁 Données

Les données incluent :

* Patients
* Coûts (Billing Amount)
* Type d’admission
* Résultats médicaux

## 🔧 Traitement des données

* Nettoyage avec Power Query
* Gestion des formats (locale US)
* Création de variables (Durée, Age Group, KPI)

## 📊 KPI développés

* Coût total
* Coût moyen
* Durée moyenne de séjour
* Taux d’urgence
* Taux d’anomalie

## 📈 Dashboard

* Vue globale
* Performance opérationnelle
* Qualité
* Analyse temporelle

## 🛠 Outils

* Power BI
* Power Query
* DAX
